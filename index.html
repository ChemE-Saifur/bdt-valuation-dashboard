4. Calculation Methodology

A. Cross-rate calculation

BDT per competitor currency = USD/BDT ÷ USD/competitor currency

B. Required exchange rate based on cumulative inflation differential

Required BDT/competitor rate today = Base BDT/competitor rate on 1 Jan 2025 × (1 + Bangladesh cumulative inflation since 1 Jan 2025) ÷ (1 + competitor cumulative inflation since 1 Jan 2025)

C. Overvaluation calculation

Overvaluation (%) = (Required BDT/competitor rate - Actual BDT/competitor rate) ÷ Actual BDT/competitor rate × 100

D. Interpretation

If Bangladesh cumulative inflation is higher than competitor inflation, BDT must depreciate against that competitor's currency to preserve real export competitiveness. If BDT does not depreciate enough, the model records BDT overvaluation.

5. Data Update Rule

  1. Keep 1 January 2025 as the fixed base date.
  2. Update current USD/BDT, USD/INR, USD/VND and USD/CNY every week.
  3. Update cumulative CPI inflation from 1 January 2025 for Bangladesh, India, Vietnam and China.
  4. Do not use only year-on-year inflation. Use cumulative inflation since the base date.
  5. Compare actual BDT depreciation with required BDT depreciation.
Important caution: The base exchange rates and cumulative inflation values must be verified from official sources. The structure is correct, but the dashboard output is only as accurate as the input data.
